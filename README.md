# markdown-converter
 LaTeX mathematical notation to plain markdown


# Usage

1.
    ```
    git clone
    ```
2.
    ```
    cd
    ```
3.
    Put the LaTeX in pre.md

4.
    Then 
    ```
    python convert.py
    ```

5. result in post.md


## To read LaTeX (pre.md)
https://www.xyzeditor.com/

## To read plaintext (post.md)
https://markdownlivepreview.com/
